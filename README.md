# 📊 Ledger — Freelancer Operations SaaS Platform

Ledger is a full-stack web application designed to simplify client CRM, billable time tracking, and automated Stripe invoicing for independent contractors and small agencies.

![Ledger Banner](https://ledger-dashboard-client-m82k.vercel.app/og-image.png)

## 🌐 Live Demo
- **URL:** [https://ledger-dashboard-client-m82k.vercel.app/](https://ledger-dashboard-client-m82k.vercel.app/)

## ✨ Key Features
- **Client CRM:** Unified client directory, active project logs, and contract storage.
- **1-Click Invoicing:** Convert logged billable hours directly into Stripe payment invoices.
- **Automated Overdue Chasing:** Surface unpaid accounts with reminder triggers.
- **Fully Responsive:** Mobile-first architecture built for seamless mobile & desktop usage.

## 🛠️ Tech Stack & Architecture
- **Frontend:** React, Vite, Modular CSS / CSS Custom Properties
- **Backend:** Node.js, Express.js (Vercel Serverless Functions)
- **Database:** MongoDB Atlas
- **Integrations:** Stripe API, Resend (Emails)
